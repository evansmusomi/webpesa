== README

This app demonstrates a simple money transfer service. To view the app check out http://wpesa.herokuapp.com

Highlights:

* Ruby version: 2.3.0

* Rails version: 4.2.6

* Authentication: Devise (Warden)

* Database: Postgres (Production) and SQlite

* Testing: RSpec, FactoryGirl, FFaker and Shoulda-Matchers

* Email service: SendGrid

* UI styling: Twitter Bootstrap
